This repository contains the supplementary information of the article entitled "Spatio-temporal modeling to identify factors associated with stunting in Indonesia using a Modified Generalized Lasso". We provided R code for this case study (20240916 Code of Data Analysis v1.R), along with the spatio-temporal data (20240916 data stunting v5.csv), information of provincial coordinates (20240915 Coordinates of Provinces.csv), and Indonesian maps (https://ipb.link/indonesian-map). Authors: Septian RAHARDIANTORO, Alfidhia Rahman Nasa JUHANDA, Anang KURNIA, 
Aswi ASWI, Bagus SARTONO, Dian HANDAYANI, Agus Mohamad SOLEH, Yusma YANTI, Susanna CRAMB.
